## Checklist

1. **Component consistency**: Are shared components used instead of one-offs?
2. **Spacing/typography**: Does it follow the design system tokens?
3. **Responsive**: Does it work on mobile, tablet, desktop?
4. **Loading states**: Do async operations show loading indicators?
5. **Error states**: Do failures show user-friendly messages?
6. **Empty states**: Do lists/tables handle zero items gracefully?
